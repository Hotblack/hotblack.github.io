PROGRAMMING CHALLENGE

  1. The Age Calculator

    Forgot how old someone is? Calculate it!

    • Store the current year in a variable.
    • Store their birth year in a variable.
    • Calculate their 2 possible ages based on the stored values.
    • Output them to the screen like so: "They are either NN or NN", substituting the values.

  2. The Temperature Converter

    Let's make a program to converts a random celsius tempatures to fahrenheit and vice versa, so we can complain about the weather with our friends oversees.

    Reminder: To convert celsius to fahrenheit, multiply by 9, then divide by 5, then add 32. To convert fahrenheit to celsius, deduct 32, then multiply by 5, then divide by 9.REMEMBER ORDER OF OPERATIONS!

    To generate a random number we use the Math.random() method. It will return a number from 0 (inclusive) to 1 (exclusive). Use that randomly generated number to create a random temperature to convert. HINT: think about how we convert a decimal into a whole integer.

    Unicode Characters: To print the degrees symbol in JavaScript, we need to use the write \u00B0 to represent the unicode character for the degress symbol.PLACE THE UNICODE CHARACTER INSIDE THE STRING

    • Store a celsius temperature into a variable.
    • Convert it to fahrenheit and output "NN°C is NN°F".
    • Now store a fahrenheit temperature into a variable.
    • Convert it to celsius and output "NN°F is NN°C."

  3. Pluralize

    Write a function pluralize that takes in two arguments, a number and a word, and returns the plural. For example:

    pluralize(5, 'cat'): '5 cats'
pluralize(7, 'turtle'): '7 turtles'

    Bonus: Make it handle a few collective nouns like "sheep" and "geese".
