More information about functions and methods

  • Functions
    • A JavaScript function is a block of code designed to perform a particular task.
      • example:

        • function convertToF(c)

        {

        var outputFahrenheit = (c * 9)/5 + 32;

        console.log(c + '\u00B0 Celsious is ' + outputFahrenheit + '\u00B0 Fahrenheit');

        }

    • A JavaScript function is executed when "something" invokes it (calls it).
      • example of how to call a function:

        • convertToF(78);

    • A function call can be empty or have one or more arguments inside
      • How many arguments depends on how many parameters were declared when the function was declared.

  • Methods
    • Simply put, A method is a special type of function that belongs to an object.
    • example:
      • Math.random();
        • MATH is the object
        • random() is the function that belongs to the object MATH
