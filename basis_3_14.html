  • More information about functions and methods

    • functions
      • A JavaScript function is a block of code designed to perform a particular task.
        • example:

          • function convertToF(c)

          {

          var outputFahrenheit = (c * 9)/5 + 32;

          console.log(c + '\u00B0 Celsious is ' + outputFahrenheit + '\u00B0 Fahrenheit');

          }

        • function is the key word
        • convertToF is the variable name of the function
        • And, inside the paranthesis are the parameters
          • In this example we have only one parameter, c. But we can have many.
      • A JavaScript function is executed when "something" invokes it (calls it).
        • example:

          • convertToF(78);

    • methods
      • Simply put, A method is a special type of function that belongs to an object.
      • example:
        • Math.random();
          • MATH is the object
          • random() is the function that belongs to the object MATH
          • The MATH object is created by Javascript. You can create your own objects and assign them to functions but we are not covering that in this class.
      • NOTE: We will not be creating our own methods or objects in this class. But we do reference them. So a little understanding is necessary.
