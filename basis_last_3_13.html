LEAVING THE WALLED GARDEN

HOW TO CODE OUTSIDE OF CODEHS

******************************************************

  • What do we need?
    • Text Editor (Notepad, Wordpad, MS Word, Sublime, Notes, etc.)
    • Web Browser
  • What goes where?
    • In this class, you will have three files:
      1. HTML (ex. index.html)
      2. CSS (ex. style.css)
      3. Javascript (ex. script.js)
    • Save them to the same folder (directory) and make sure the file extensions (ie .js, .css, .html) are present!
  • How to link the HTML file to the CSS and Javascript files
    • CSS
      • Because we create our own CSS file, we must specify where the file is located inside our own HTML file
      • Inside the head tags we add the following:
        • <link="stylesheet" href="style.css">
          • rel refers to the type of file we are linking to. In this case we are linking a stylesheet.
          • href refers to the location and name of the file we are linking to. In this case, style.css is located in the same folder (directory) as the html file.
          • More Info about linking
    • Javascript
      • Javascript is slightly different than CSS
      • Instead of using the link tags, we add the location of the javascript file to the src attribute in the script tags
      • So we add the following to the head tags in our html file:
        • <script src="script.js">
  • What now?
    • We will continue to use CODEHS
    • It is too convenient to give up this late in the year
    • However, if you do own a laptop, I would highly recommend working outside of CODEHS
    • If you continue with Computer Science or programming in general, you will not be using CODEHS. Best to get used to the real tools before then!
